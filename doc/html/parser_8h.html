Detailed Description

Prototypes, macros, enumerators and global variables...

The COMMANDS enumerator contains a member for each command supported by the language. The memebers of the enumerator appear in the same order as the corresponding commands, in the commands table (see parser.c). The last member of the enumerator is used for assignments (some_variable = some_value) which do not use some special command. I other words, CMD_ASSIGNMENT corresponds to no entry in the commands table!
