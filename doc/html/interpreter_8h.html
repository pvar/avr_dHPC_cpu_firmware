Detailed Description

System specifications, global variables and function prototypes.

Most of the macros in this file play a decisive role in what this system can achieve! The program memory size determines the maximum size of user programs, while the stack size determines the maximum number of nested function calls.

void interpreter ( void  )

The interpreter main loop.

This function examines user input and determines if the last line entered was a command or a program line. In the first case, the command is executed immediately. In the second case, the line read is merged with the rest of the program and into appropriate position.
