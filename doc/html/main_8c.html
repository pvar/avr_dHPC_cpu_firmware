void get_line ( void  )

Get line(s) from EEPROM, SERIAL or STDIO.

Get one or more lines, depending on the selected input. If the system configuration (sys_config) dictates that new data should be received from EEPROM or SERIAL (this happens when loading programs from either source) the function will keep getting lines until a NULL character is received. If new data is to be received from STDIN (the user himself) the fuinction will only get a single line.
